# PHP MVC Book Management System

## Overview
This project is a Book Management System built using PHP with MVC architecture.

## Features
- Add, update, delete books
- MVC structure for better code organization
- Database integration using MySQL
- Clean and maintainable code

## Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript

## Purpose
This project demonstrates backend development skills and understanding of MVC architecture.

## Note
This is a sample/demo project created for learning and demonstration purposes.
